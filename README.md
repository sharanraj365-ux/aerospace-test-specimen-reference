# Aerospace Test Specimen Standards — Reference Document

## Overview
A consolidated technical reference document covering dimensional 
requirements, material specifications, and test purpose definitions 
for aerospace corrosion and mechanical test specimens.

Compiled from publicly available ASTM, SAE AMS, and European 
aerospace (EN) standards for incoming inspection, manufacturing 
scope review, and RFQ preparation activities.

---

## Standards Covered

| Standard | Test Type | Key Application |
|---|---|---|
| ASTM B117 | Neutral Salt Spray | Corrosion resistance — metals & coatings |
| ASTM B368 | CASS Test | Accelerated corrosion — decorative coatings |
| ASTM F1110 | Sandwich Corrosion | Aircraft maintenance chemical evaluation |
| ASTM G50 | Atmospheric Corrosion | Long-term outdoor exposure testing |
| ASTM G69 | Corrosion Potential | Aluminum alloy electrochemical behavior |
| ASTM G48 | Pitting & Crevice | Stainless steel & Ni-alloy corrosion |
| AMS-C-26074 | Ni Plating Corrosion | Electroless nickel plating qualification |
| ASTM F482 | Chemical Immersion | Aircraft maintenance chemical corrosiveness |
| ASTM C633 | Thermal Spray Bond | Coating adhesion/cohesion strength |
| ASTM F519 (1a.1 & 1a.2) | Hydrogen Embrittlement | Plating process qualification |
| EN 6072 | Constant Amplitude Fatigue | S-N curve generation — aerospace metals |
